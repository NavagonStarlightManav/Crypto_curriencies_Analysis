# ₿ Cryptocurrency Market Analysis Dashboard

An interactive Power BI dashboard designed to analyze historical cryptocurrency market performance through market capitalization, trading volume, and price movement trends. The project demonstrates an end-to-end business intelligence workflow—from raw data preparation to interactive dashboard development.


## Project Overview

This project transforms historical cryptocurrency market data into an interactive analytical dashboard using Power BI. It enables users to monitor market trends, compare cryptocurrency performance over time, and explore key financial indicators through dynamic visualizations.

The dashboard focuses on delivering actionable insights while maintaining a clean, intuitive, and business-oriented design.


## Dataset

The analysis is based on historical cryptocurrency market data containing over **37,000 records**.

### Features

- Cryptocurrency Name
- Symbol
- Date
- Open Price
- Close Price
- High Price
- Low Price
- Trading Volume
- Market Capitalization

## Project Workflow

### 1. Data Collection

- Imported historical cryptocurrency dataset into Power BI.

### 2. Data Preparation

- Checked data quality.
- Converted data types.
- Prepared date fields for time-series analysis.

### 3. Data Modeling

- Structured the dataset for efficient reporting.
- Created calculated fields and aggregated measures where required.

### 4. Dashboard Development

- Designed interactive KPI cards.
- Built time-series visualizations.
- Added slicers for dynamic filtering.
- Implemented page navigation for improved user experience.

### 5. Insight Generation

- Evaluated market capitalization trends.
- Compared opening and closing prices.
- Analyzed high and low price fluctuations.
- Monitored trading volume over time.
